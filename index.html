Карбонат натрия Na₂CO₃ получают аммиачным способом — сложным замкнутым циклом, в котором аммиак регенерируется, а CO₂ используется повторно.

Na₂CO₃ NaHCO₃ Аммиачный способ Замкнутый цикл 5 основных реакций 4 основные стадии
Na₂CO₃

Кальцинированная сода

Бесцветные кристаллы, гигроскопична. Применяется в производстве стекла, мыла, варке целлюлозы, очистке нефтепродуктов.

NaHCO₃

Питьевая сода

Хорошо растворима в воде. Применяется в хлебопечении, производстве напитков, медицине и огнетушащих составах.

// 05

Принципиальная схема

Замкнутый цикл. Аммиак не расходуется — он регенерируется из маточного раствора. CO₂ из кальцинации и известковых печей повторно используется в карбонизации.

1. Добыча и очистка рассола NaCl

Рассол получают выщелачиванием каменной соли. Очищают от Ca²⁺ и Mg²⁺ (реакции 6, 7). T = 17°C.

2. Аммонизация (абсорбция NH₃)

Очищенный рассол насыщается аммиаком из отделения дистилляции. T ≈ 60°C. Два абсорбера с промежуточным охлаждением до 30°C.

3. Карбонизация → осадок NaHCO₃

Аммонизированный рассол насыщается CO₂ (реакция 3). NaHCO₃ выпадает в осадок. Серия из 4 колонн, 3 работают как осадительные.

4. Фильтрация

Барабанные вакуум-фильтры отделяют кристаллы NaHCO₃ от маточного раствора NH₄Cl.

5. Кальцинация → Na₂CO₃ (готовый продукт)

2NaHCO₃ → Na₂CO₃ + H₂O + CO₂ (реакция 5). CO₂ возвращается на карбонизацию.

6. Дистилляция → регенерация NH₃

Маточный раствор NH₄Cl + Ca(OH)₂ → NH₃ (реакция 4). Аммиак возвращается на аммонизацию. CaO и CO₂ получают из обжига известняка.

// ДЕФЕКТ 1

Недопал

Возникает при недостаточной температуре обжига или слишком крупных кусках материала. CaCO₃ не полностью разлагается.

// ДЕФЕКТ 2

Пережог

Температура выше 1250°C. CaO переходит из активной формы в неактивную и перестаёт реагировать с водой — гашение невозможно.

// 07

Шахтная известковая печь

Шихта (смесь известняка с коксом) движется сверху вниз, газы — снизу вверх (принцип противотока). Печь делится на три тепловые зоны.

↓
Зона подогрева Верхняя часть
Подсушка сырья и топлива. Нагрев шихты горючими газами, движущимися противотоком из зоны обжига. На границе зон — начало разложения CaCO₃.
800–1000 °C на границе
⚡
Зона обжига Средняя часть
Горение топлива + разложение CaCO₃ → CaO + CO₂. В нижней части зоны горение продолжается, тепло расходуется на нагрев воздуха из зоны охлаждения.
1100–1200 °C в конце зоны
↑
Зона охлаждения Нижняя часть
Охлаждение готового CaO воздухом, поступающим снизу. Воздух нагревается и поднимается в зону обжига.
↓ Охлаждение CaO
Выгрузка и гашение. Образовавшийся CaO непрерывно выгружается ковшевым транспортёром → бункеры → гаситель. Гашение: 20–25 минут, температура воды 60–80°C. Температура воды влияет на скорость и полноту гидратации.

Аммонизация рассола

Назначение: получить аммонизированный рассол с необходимой концентрацией NH₃ и передать его в отделение карбонизации.
Температура в абсорбере
~60 °C

30 °C
Число ступеней абсорбции
2
С промежуточным охлаждением

Ключевая реакция при аммонизации — образование карбамата аммония, который снижает равновесное давление NH₃ над раствором и ускоряет поглощение:

—
2NH₃ + CO₂ → NH₂COONH₄ (карбамат аммония)
Поток рассола — 75%

Через промыватели фильтров

Промыватель воздуха фильтров (4) → промыватель газа (7). Поглощается NH₃ из воздуха вакуум-фильтров и из отходящих газов карбонизации.

Поток рассола — 25%

Через промыватель абсорбции

Промыватель газа абсорбции (1). Поглощается NH₃ из выхлопных газов отделения абсорбции + часть CO₂.

Оба потока сходятся → два абсорбера (13 и 14) последовательно, газ движется противотоком к рассолу. После первого абсорбера рассол охлаждается с 65°C до 30°C в оросительном холодильнике, затем поступает во второй абсорбер.
// 09

Карбонизация

Назначение: насыщение аммонизированного рассола CO₂ → кристаллизация и осаждение NaHCO₃ по реакции (3).

Процесс протекает ступенчато:

I
2NH₃ + CO₂ → NH₂COONH₄
CO₂ реагирует с NH₃ → карбамат аммония
II
NH₂COONH₄ + H₂O → NH₄HCO₃ + NH₃
Гидролиз карбамата → перенасыщенный раствор HCO₃⁻
III
NH₄HCO₃ + NaCl → NH₄Cl + NaHCO₃↓
Из перенасыщенного раствора кристаллизуется NaHCO₃
Конструкция колонны

Барботажный аппарат

Состоит из чугунных царг с колпачковыми тарелками (один колпачок). В нижней части — встроенные холодильники.

Два ввода газа

Смешанный + печной

Низ: смешанный газ 70–80% CO₂.
Середина (10–12 м): печной газ из известковых печей, 40% CO₂.

// 10

Фильтрация суспензии

Назначение: отделение кристаллов NaHCO₃ от маточного раствора NH₄Cl с помощью барабанных вакуум-фильтров.
После фильтрации → продукт

Осадок NaHCO₃

Снимается с барабана вакуум-фильтра, поступает на транспортёр → отделение кальцинации (реакция 5).

После фильтрации → рецикл

Маточный раствор NH₄Cl

Из сборника насосом подаётся в отделение дистилляции для регенерации NH₃ (реакция 4).

Схема работы отделения фильтрации:

Суспензия из карбонизационных колонн → ёмкость (1)

Подаётся чуть больше, чем нужно фильтру, для поддержания постоянного уровня в корыте.

Корыто вакуум-фильтров (2)

Количество регулируется дроссельными заслонками. Барабанные вакуум-фильтры разделяют суспензию на осадок и фильтрат.

Сепаратор (4)

Смесь воздух + фильтровая жидкость разделяется. Жидкость → сборник (6) → насос (5) → дистилляция. Воздух → насос (10) → промыватель (11) → атмосфера.

Вакуумный сборник (7) с мешалкой

Принимает избыток суспензии из корыта. Мешалка препятствует оседанию. Насосом возвращает суспензию в ёмкость (1).

Промывка осадка и тканей

Раствор из сборника (9) используется для промывки осадка NaHCO₃ и регенерации фильтрующих тканей. Сжатый воздух подаётся воздуходувкой.
